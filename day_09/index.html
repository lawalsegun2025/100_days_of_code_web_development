HTML Summary

HTML (HyperText Markup Language) is used to define our page content, structure and meaning. You don not use it for styling purposes. Instead, use CSS for styling.

  • HTML uses "elements" to describe (annotate) content.
  • HTML elements typically have an opening tag, content and then a closing tag.
  • You can also have void (enpty) elements like images.
  • YOu can also cinfigure elements with attributes.
  • There is a long list of availabel elements but you will gain experience over time, no worries.
